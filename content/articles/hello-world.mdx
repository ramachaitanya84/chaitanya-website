## Getting Started

To add more articles, simply create a new `.mdx` file in `content/articles/` with frontmatter:

```yaml
---
title: "Your Title"
date: "2025-02-22"
excerpt: "Brief description"
slug: "your-slug"
---
```

Then write your content in Markdown (or MDX) below the frontmatter.
